# Two Stage Op Amp

#### Aim : 

To design a two stage opamp on Cadence Virtuoso at 90nm technology node for the following specifications- 
- DC gain = 50dB
- GBW = 30MHz
- PM >= 60 deg
- Slew Rate >= 20V/$\mu$S
- $ICMR^+$ = 1.6V
- $ICMR^-$ = 0.8V
- $C_L$ = 2pF
- $V_{dd}$ = 1.8V
- Process = 90nm
